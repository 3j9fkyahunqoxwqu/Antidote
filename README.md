## Downloads

Clone repo `git clone --recursive https://github.com/dvor/Antidote.git` and open `Antidote.xcworkspace` file with Xcode 5+.

## Features

-  1v1 messaging
-  typing notification
-  emoticons
-  spell check
-  reading/scanning Tox ID via QR code
